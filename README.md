# AI Interview Calendar
Календарь для планирования и отслеживания собеседований с автоматическим анализом вакансий через ИИ, 
цепочками этапов, браузерными уведомлениями и удобным интерфейсом.

## 🚀 Основные возможности
- **Календарь событий** – просмотр запланированных собеседований по месяцам, переход на детальную страницу дня и события.
- **Управление событиями** – создание, удаление, перенос даты, добавление следующего этапа (цепочка собеседований).
- **Анализ вакансий с AI** – при создании события автоматически парсится страница вакансии (hh.ru) 
и с помощью ИИ проводится анализ вакансии, выдавая основные моменты, топ вопросов для подготовки.
- **Выбор ИИ** - можно легко переключить ИИ в .env (на данный момент реализован только Yandex AI, позже будет OpenAI)
- **Цепочки этапов** – каждое событие может быть родительским или дочерним, отображается полная история собеседований.
- **Уведомления** – браузерные напоминания за 10 минут до события (работают на всех страницах, пока открыт браузер).

## 🛠 Технологии
- Backend: Laravel 11 (PHP 8.2)
- Frontend: Blade, JavaScript (ES6+), Vite, Axios, Браузерные уведомления через Notification API
- База данных: PostgreSQL 16
- Очереди: Redis
- Сервер: Nginx, PHP-FPM
- Контейнеризация: Docker, Docker Compose

## 🐳 Запуск через Docker
- Предварительные требования Установленные Docker и Docker Compose
- Настройте .env
- Соберите и запустите контейнеры `docker compose up -d`
- Миграции выполнятся автоматически
- Откройте приложение http://localhost:8080

## 📁 Структура проекта (основные папки)
* `app/`
  * `Console/`
    * `Commands/` 
      * `RefreshVacanciesData.php` - команда artisan для обновления анализа по вакансиям
  * `DTO/` Data Transfer Objects
  * `Enums/`
    * `EventStatus.php` - Статусы событий
  * `Http/` 
    * `Controllers/` Контроллеры
    * `Requests/` Валидация
  * `Jobs/`
    * `ParseVacancyJob.php` Парсинг вакансии
  * `Models/` Модели
  * `Repositories/` Репозитории (работа с БД)
  * `Services/`
    * `AiProviderFactory.php` - Фабрика определения ИИ 
    * `EventService.php` - Бизнес логика событий
* `docker/` Dockerfile, entrypoint.sh, nginx/default.conf
* `resources/`
  * `css/ ` Стили
  * `js/` JS скрипты
  * `views/` Blade-шаблоны
* `docker-compose.yml` Оркестрация контейнеров
